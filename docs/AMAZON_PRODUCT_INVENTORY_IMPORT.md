# Amazon Product Inventory Import

Amazon Product Inventory Refresh accepts All Listings, category XLSM/XLSX, product TXT/TSV, supporting CSV, multiple files and ZIP. All Listings supplies identity/status; catalogs supply attributes/images. Reference sheets are inert and VBA/macros/formulas never execute.

Image/title-only data cannot create products. Amazon consignments normally require only the shipment quantity file.
